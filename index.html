Repositorio de normativa de ciberseguridad

Este sitio aloja los textos oficiales de normativas de ciberseguridad utilizadas como fuente documental por el agente NormatIA.

La información contenida en este repositorio se utiliza exclusivamente para análisis técnico‑normativo y contraste documental.

Normativas disponibles

  • Cyber Resilience Act (CRA)
    Reglamento europeo sobre requisitos de ciberseguridad para productos con componentes digitales.
    📄 Ver texto oficial del CRA (PDF)

  • IEC 62443
    Norma internacional para la ciberseguridad en sistemas de automatización y control industrial.
    📄 Ver documentos IEC 62443

  • Esquema Nacional de Seguridad (ENS)
    Marco normativo español para la seguridad de sistemas de información.
